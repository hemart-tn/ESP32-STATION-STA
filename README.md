# 📡 ESP32 Station (STA) + Web Dashboard

Proyek IoT menggunakan ESP32 dalam mode **Station (STA)** yang terhubung ke WiFi rumah dan menampilkan data sensor melalui **web dashboard realtime**.

---

## ⚙️ Fitur
- Mode STA (terhubung ke WiFi)
- Web server di ESP32
- Monitoring realtime via browser
- Interface sederhana (HTML, CSS, JS)

---

## 🚀 Cara Menjalankan
1. Isi SSID & Password WiFi di kode
2. Upload ke ESP32
3. Buka Serial Monitor
4. Akses IP ESP32 di browser

---

## 🌐 Akses
Gunakan IP dari Serial Monitor: http://192.168.x.x

---

## ⚠️ Catatan
- ESP32 dan perangkat harus di jaringan yang sama
- Pastikan koneksi WiFi stabil
